# Security Policy

Copyright (c) StagePlane contributors.

## Architectural intent

This file documents how to report vulnerabilities in the public `stagectl` release-distribution surface.

## Ownership boundary

This repository owns installer and binary-publication security. Security issues in the private controller source, license server, or infrastructure modules should still be reported through the same StagePlane security channel.

## Non-responsibilities

This repository does not accept customer secrets, licenses, cloud credentials, or private infrastructure configuration.

## Reporting

Report vulnerabilities to:

```text
security@stageplane.io
```

Do not open public GitHub issues for suspected vulnerabilities.
